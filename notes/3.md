# 寻找重复的行

类似于 Unix 的 `uniq` 实用工具，使用了 `bufio` 包。

[dup1](../codes/ch1/dup1.go)

如何运行这个程序呢？可以新建一个文件 `data.txt`，其中写入一些文字，然后运行：

```sh
$ cat data.txt | ./dup1
```

| 符号 | 解释 | 
| --- | --- |

| `%T` | 任意数值的类型 |
| `%%` | 百分比字面量 |
